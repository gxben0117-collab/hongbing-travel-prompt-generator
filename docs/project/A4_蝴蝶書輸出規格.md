# A4 蝴蝶書輸出規格

Date: 2026-05-19

## 用途

本規格用於「藍寶原圖」批次生成厚板蝴蝶書、寫真書、A4 直式印刷頁面。

目標不是單張社群海報，而是可直接放入 21 x 29.5 cm 版面的印刷照片。

## 尺寸與比例

- 輸出尺寸：2416 x 3408 px
- 版面比例：A4 portrait vertical / 21:29.5
- 方向：直式
- 版面要求：保留安全邊界，不要讓頭、腳、裙擺、手、道具貼邊

## 固定咒語補充

每次 A4 蝴蝶書輸出都應補上：

```text
高級商業攝影質感、時尚雜誌封面感、電影級打光、細膩皮膚質感、清楚五官、超高解析度、超高細節。
A4 portrait vertical, 21:29.5 print layout, leave extra 12% safe margins on all sides, subject centered vertically and horizontally, face in upper-middle third, feet and dress inside frame, no important details near edges.
```

## 構圖規則

保留三種構圖隨機：

- 時裝全身距離
- 全身大景
- 低角度女王

暫停使用：

- 妝容特寫

原因：妝容特寫容易讓臉、頭髮、肩膀貼邊或被裁切，不適合厚板蝴蝶書印刷安全邊界。

## 必要負面限制

咒語應避免：

```text
close-up crop, cropped head, cropped hair, cropped feet, cropped dress, face too close to border, important details near edge, extreme wide angle, hidden face, distorted anatomy, watermark
```

## 後續旅拍程式修改方向

後續修改紅兵旅拍大片生成器時，應新增或調整一個「A4 蝴蝶書 / 印刷安全版」輸出模式：

- 預設尺寸提示為 2416 x 3408
- 顯示 A4 21:29.5 印刷版提示
- 啟用 12% 安全邊界咒語
- 三選一構圖，不包含妝容特寫
- 保留人物完整頭、腳、裙擺與主要道具
- 風格可以隨機，但構圖安全優先於戲劇張力
