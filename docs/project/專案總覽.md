# 紅兵旅拍大片生成器

## 專案用途

這是一個旅拍大片風格的咒語產生器，用來整理角色、場景、服裝、光影、鏡頭語言與輸出提示詞。

## 目前狀態

- 專案由原始 HTML 檔建立。
- 目前保留單頁應用形式。
- 主要功能集中在 `index.html`。
- 已建立基本專案文件與開發紀錄流程。

## 主要檔案

- `index.html` - 主要工具頁面。
- `README.md` - 專案入口說明。
- `package.json` - 本機預覽設定。
- `docs/` - 文件與開發紀錄。
