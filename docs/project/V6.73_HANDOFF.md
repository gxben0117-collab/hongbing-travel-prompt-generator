## Pending A4 Photobook Mode

The Lanbao batch work established a print-safe A4 photobook direction for later app changes.

Reference file:

- `docs/project/A4_蝴蝶書輸出規格.md`

Key decisions:

- Use `2416 x 3408 px` for A4 portrait / 21:29.5 output.
- Add commercial photography and magazine-cover quality wording.
- Add 12% safe margins, centered subject, face in upper-middle third, no important details near edges.
- Use only three random composition modes: `時裝全身距離`, `全身大景`, `低角度女王`.
- Exclude `妝容特寫` for print-safe photobook batches because it frequently crops faces/hair too close to page edges.
