# 紅兵旅拍大片生成器文件中心

目前文件版本：`V1.00`

## 文件索引

- `project/專案總覽.md` - 專案用途、主要檔案、目前狀態。
- `process/文件流程.md` - 每次更新時要怎麼記錄、怎麼新增版本檔。
- `版本規則.md` - 版本號累積規則與命名方式。
- `development-log/V1.00.md` - 初始建立紀錄。

## 更新原則

每次更新都新增一份新的開發紀錄檔，不覆蓋舊檔。

範例：

- `development-log/V1.00.md`
- `development-log/V1.01.md`
- `development-log/V1.02.md`
