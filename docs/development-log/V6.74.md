# V6.74

Date: 2026-05-19

## 變更

依使用者要求，從「風格範例分類總覽」移除以下分類，並連同各分類底下的風格範例一起移除：

- `indian`：異域宮廷
- `nuwa_series`：女媧神話
- `mazu_series`：媽祖神話
- `dramanovel`：劇本女主
- `modern`：摩登封面
- `citywalk`：城市電影

## 實作方式

- 從 `index.template.html` 移除上述分類按鈕與既有風格卡片。
- 在 `src/data.js` 增加 `REMOVED_STYLE_CATEGORIES_20260519`，於 `ALL_IDS` 產生前刪除上述分類的 preset，避免隨機、補卡、自動系列再出現。
- 更新主分類映射，避免上述分類仍出現在分類群組中。
- 重建 `dist/index.html`，並同步到 `index.html` 與 `versions/index_v6.74.html`。

## 驗證

- `node --check src/data.js`
- `node --check src/app.js`
- `npm.cmd run build`
- 驗證資料層上述 6 個分類的 preset 數量皆為 `0`。
- 驗證 `index.template.html`、`index.html`、`dist/index.html`、`versions/index_v6.74.html` 沒有上述 `data-cat` 卡片殘留。
