## 實作方式

- 更新 `REMOVED_STYLE_CATEGORIES_20260519`，新增 `studio`、`travel`、`world_culture`、`myth_angel`。
- 從 `index.template.html` 移除這些分類按鈕與風格卡片。
- 從分類群組移除這些分類，避免 UI 再顯示空分類。
- 重建 `dist/index.html`，同步到 `index.html` 與 `versions/index_v6.75.html`。
