# V6.78

Date: 2026-05-20

## Update

This release starts the first actual pruning pass for travel-mode image-reference examples.

Goal of this pass:

- reduce high-overlap `img_ref` variants
- keep overall category breadth intact
- remove low-priority examples that read too similarly from a user perspective

## Removed In This Pass

Removed two highly overlapping `img_ref` presets:

- `ref_beauty_life_cover`
- `ref_afternoon_tea_salon`

## Why These Two

Both examples leaned heavily toward:

- lifestyle / magazine mood
- refined indoor editorial styling
- soft luxury / dessert-table / salon-adjacent visual language

Compared with the remaining reference set, these two had the highest user-perceived overlap relative to their distinctiveness.

The current library still retains enough:

- editorial references
- luxury travel mood
- bridal references
- lifestyle elegance

without these two variants.
